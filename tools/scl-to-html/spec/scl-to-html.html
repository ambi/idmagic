Scenarios 7

SCL 3.0 の全 section を表示できる

Main success
  1. 全 SectionKind を持つ SclDocument を Render に渡す
  2. 各 section と所有要素の局所契約が HTML に表示される

authorization を policy と action の近くに表示する

Readerscl-to-html/scenario/authorization を policy と action の近くに表示する
Main success
  1. protected interface と authorization を持つ HTML を開く
  2. policy、principal、resource と interface access を相互に辿れる

flow を navigation diagram として表示する

Readerscl-to-html/scenario/flow を navigation diagram として表示する
Main success
  1. flows を持つ HTML を開く
  2. view と transition が SVG diagram と detail card に表示される

意味参照を安定 anchor へ移動できる

Readerscl-to-html/scenario/意味参照を安定 anchor へ移動できる
Main success
  1. scenario 内の既知 model、interface、event 名を選択する
  2. 対応する定義 card へ移動する

user text は escape される

Maintainerscl-to-html/scenario/user text は escape される
Main success
  1. HTML 特殊文字を含む SclDocument を Render に渡す

JavaScript 無効時も全内容を読める

Readerscl-to-html/scenario/JavaScript 無効時も全内容を読める
Main success
  1. JavaScript を無効にして HtmlDocument を開く
  2. 全 tab の内容が縦に表示される

work item の進捗と completion を表示する

Readerscl-to-html/scenario/work item の進捗と completion を表示する

  1. Plan、Tasks、completion を持つ ChangeEntry の HTML を開く
  2. task progress と completion summary が同じ card に表示される
